12 Cap Table Cleanup Checklist

P0 - Before Fundraising

  • ☐ Verify entity structure (Delaware C-Corp preferred for VC)
  • ☐ Confirm all equity allocations in Carta / Pulley / spreadsheet
  • ☐ Ensure IP assignment agreements signed by all contributors
  • ☐ Verify founder vesting schedule and cliff status
  • ☐ ESOP pool established and 409A valuation obtained
  • ☐ Operating agreement / bylaws current
  • ☐ Board of directors established (even if 1 person)

P1 - Before Closing

  • ☐ 409A valuation report (for option strike price)
  • ☐ Equity incentive plan (EIP) approved by board
  • ☐ SAFE / convertible note cleanup (if any)
  • ☐ Advisory agreements drafted with equity terms
  • ☐ Employee offer letters include equity grant language
  • ☐ State franchise taxes current
  • ☐ Cap table management tool selected (Carta, Pulley, AngelList)
